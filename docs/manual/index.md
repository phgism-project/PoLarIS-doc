# PoLarIS Documentation

PoLarIS 是一个面向极地冰盖模拟与冰冻圈研究的冰盖模式。本网站用于维护模型文档、记录可复现实验、展示典型案例，并为用户和开发者提供统一入口。

## 文档入口

- [快速开始](quickstart.md)：从准备环境到完成一次测试运行。
- [安装与编译](installation.md)：软件依赖、编译选项与常见问题。
- [输入数据](input-data.md)：地形、气候强迫、初始场与边界条件。
- [参数配置](configuration.md)：配置文件结构与关键参数说明。
- [运行流程](workflow.md)：实验组织、运行命令与批处理建议。
- [输出说明](outputs.md)：输出变量、文件格式与后处理方法。

!!! note "维护建议"
    每个功能模块、参数组和案例实验都建议单独维护为一个 Markdown 页面。这样可以避免单页过长，也方便多人协作。
