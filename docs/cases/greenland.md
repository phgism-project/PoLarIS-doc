# 格陵兰冰盖案例

## Overview

该案例用于记录 PoLarIS 在格陵兰冰盖区域的模拟流程、输入数据和结果诊断。

## Input Data

| 数据 | 来源 | 说明 |
| --- | --- | --- |
| 地形 | 待补充 | 床面和表面高程 |
| SMB | 待补充 | 表面质量平衡 |
| 温度 | 待补充 | 热边界条件 |

## Configuration

```yaml
experiment:
  name: greenland_baseline
domain:
  region: greenland
```

## Results

建议展示：

- 冰厚空间分布
- 表面速度
- 冰盖总体积时间序列
- 接地线或冰缘变化
